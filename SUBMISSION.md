# DoraHacks Submission Guide: APEX.AI

Use the answers below to ensure your submission hits every judging criteria at the highest level.

### 📝 One-Line Pitch
> "APEX.AI is an autonomous agentic immune system for Mantle that identifies and logs malicious bots in real-time onto a global on-chain registry."

---

### 🔍 Key Submission Questions

**1. Which data sources does your project use?**
"APEX.AI utilizes real-time JSON-RPC data from Mantle Mainnet to scan every transaction. It cross-references this with a custom-built heuristic intelligence engine and maintains state via our Global Bot Registry contract on Mantle Sepolia."

**2. What role does AI play?**
"AI acts as the 'Inspector' and 'Signer.' The engine uses pattern-recognition intelligence to distinguish between human actors and machine-driven exploits. Furthermore, the system operates as an 'Autonomous Agent,' signing transactions on-chain independently to ensure 24/7 protection of the network without user friction."

**3. How does it generate verifiable value on Mantle?**
"APEX.AI creates a transparent 'Wall of Shame' for malicious actors. By registering bot addresses on-chain, we enable DeFi protocols (especially RWA protocols like USDY) to protect their liquidity from toxic order flow. This increases yield for real human investors and reinforces Mantle's reputation as a secure, agent-friendly ecosystem."

**4. How is it realized on Mantle?**
"The project is fully realized through a hybrid architecture: scanning high-speed Mantle Mainnet traffic and logging results to Mantle Sepolia. We leverage Mantle’s low fees to make continuous, autonomous on-chain logging economically viable for a security agent."
